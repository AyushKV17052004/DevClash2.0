🚀 AI Companion – Your Smart Study & Productivity Partner

AI Companion is an all-in-one intelligent assistant designed to enhance learning, productivity, and daily workflows. Built with modern web technologies and powered by advanced AI, it helps users interact, learn, and organize tasks more efficiently.

✨ Features
🤖 AI Tutor – Ask questions and get instant, clear explanations
🧠 Spaced Repetition System – Improve memory with smart revision scheduling
📅 Study Planner – Organize your study sessions and stay consistent
📊 Dashboard – Track progress and manage activities in one place
🎥 Video Summarizer – Convert long videos into concise summaries
🖼️ Image to Text (OCR) – Extract text from images using AI
🎙️ (Optional) Audio input support for interactive learning
🛠️ Tech Stack
Frontend: React + Vite
Backend: Node.js + Express
AI Integration: Gemini API
Styling: CSS / Tailwind (if used)
